IS Bookmarks v1.1

Tutorial:
Create and update Illegal Services indexed websites in a single bookmark.
The extension is supposed to create a bookmark called "Illegal Services" in your bookmarks bar.
How it works:
Once installed it creates an "Illegal Services" bookmark and every time you start your browser it updates it.
Know Bugs:
  • You cannot customize the location of the bookmarks folder. It can only be in the taskbar view.
  • Firefox sometimes has some entries in reverse order.
  • If you are using Default Bookmark Folder at the same time, Illegal Services Bookmarks will create all bookmarks in the base folder, resulting in a huge mess.
